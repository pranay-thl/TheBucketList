Demo 3: A great responsive tool

 

Resize your window to see how the main menu at the top of the page is hiddened, and how it is converted to a side menu that you can expand with a button.

 

This responsive solution is carried out by the super panel: the panel can automatically show/hide elements in the page when the screen reaches a preset breakpoint.

 

This is set by the {showMode: "yn, 1000, nn"} setting in the super-panel.js.

 

For details please visit Full documentation of the Super Panel
